Explain the concept of Responsive Design. What does the term mean?

Responsive design is a GUI based design approach which helps in creating the webpage. It allows the webpage to have flexible layout, images and CSS media queries. The main objective of the responsive web design is to identify the user screen size and adjust the webpage layout accordingly.

How does responsive design relate to web technologies?

The responsive design includes three core principles. These are Fluid Grid System, Fluid Image Use and Media quires. Here, the Fluid Grid system refers to the page layout in which all the elements occupy the same percentage of space irrespective of the user screen size. The fluid Image use refers to the images size. The fluid image user helps in adjusting the image size as per user screen size. And media queries are filters used to detect the browsing device’s dimension and make the design look clear. It can also alter the webpage layout to meet the specific conditions.
